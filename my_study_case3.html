A/B Testing

1. A/B Testing Steps

1.1 Customer Funnel

Funnel analysis for each action steps.

1.2 Define Metric

Target increase metrics such as Click-through Rate, Conversion Rate, Renewal Rate, Bounce Rate, Average Retention.

1.3 Form Hypothesis

Hypothesis Test, Null Hypothesis.

1.4 Formulate Test Plan

Make a plan to define 'good'.

1.5 Create Variation

Directly compare a variation against a current experience, including data collection in this step.

1.6 Run Experiment

Choose significance level, statistical power and practical significance level.

1.7 Analyze Test Result

Sanity check, single metrics evaluation and cross checking by using differrent methods.

1.8 Conclusion

Do you understand the change?

Do you want to launch the change?

If it's Yes, is it worth to launching?
